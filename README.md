# requirement
* docker, docker-compose (ex. Docker for Mac)
* **"psql"** command . (on Mac OS, "brew install postgresql")
* JDK8 or later

# usage
0. brew install postgresql
1. sh setup.sh
2. ./gradlew run -p pj-web
3. see http://localhost:8080
